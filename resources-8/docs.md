# User's friends list

## Description:

This app display list of user's friends using VK API

## Launch example:

### Input:

```
> python task-8.py --userid *example: andreykaf1* --token *example: 12487624fhskc12482*
```

### Output (in response.txt file):

```
Александров Александр
Артемов Артем
Иванов Иван
Николаев Николай
Петров Петр
```
